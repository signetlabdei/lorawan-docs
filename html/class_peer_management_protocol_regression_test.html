Detailed Description

Peering Management Protocol regression test.

Initiate scenario with 2 stations. Procedure of opening peer link is the following: 

* |----------->|  Beacon
* |<-----------|  Peer Link Open frame 
* |----------->|  Peer Link Open frame
* |----------->|  Peer Link Confirm frame
* |<-----------|  Peer Link Confirm frame
* |............|
* |<---------->|  Other beacons
*
