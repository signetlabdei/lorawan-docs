Detailed Description

This system test program creates different test cases with a single eNB and several UEs, all having the same Radio Bearer specification. In each test case, the UEs see the same SINR from the eNB; different test cases are implemented obtained by using different SINR values and different numbers of UEs. eNb and UEs are configured to use the secondary carrier and the component carrier manager is configured to split the data equally between primary and secondary carrier. The test consists of checking that the throughput obtained over different carriers are equal within a given tolerance.
