Detailed Description

Check the number of times that PktsAcked is called.

Set a custom congestion control class, which calls PktsAckedCalled each time the TCP implementation calls PktsAcked.

The checks are performed in FinalChecks: the number of bytes acked divided by segment size should be the same as the number of segments passed through PktsAcked in the congestion control.
