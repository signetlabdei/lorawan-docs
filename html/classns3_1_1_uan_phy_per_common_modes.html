Detailed Description

Packet error rate calculation for common tx modes based on UanPhyPerUmodem

Calculates PER for common UanTxMode modulations, by deriving PER from the BER taken from well known literature's formulas.

Calculate the Packet ERror probability based on SINR at the receiver and a tx mode.

This implementation calculates PER for common UanTxMode modulations, by deriving PER from the BER taken from literature's formulas.
