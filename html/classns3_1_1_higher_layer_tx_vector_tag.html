Detailed Description

This tag will be used to support higher layer control DataRate and TxPwr_Level for transmission. If the high layer enables adaptable mode, DataRate will be the minimum allowable value and TxPwr_Level will be the maximum allowable value for transmission. If the higher layer does not enable adaptable parameter, the DataRate and TxPwr_Level will be actual values for transmission. However, if this tag is not used and inserted in the packet, the actual DataRate and TxPwr_Level for transmission will be determined by MAC layer.
