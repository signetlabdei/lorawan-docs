Detailed Description

This test case tests channel access assignments which is done by StartSch and StopSch method of WaveNetDevice.

channel access assignments include ContinuousAccess, ExtendedAccess, and AlternatingAccess. The results of this test case depend on the implementation of ChannelScheduler In this case, the results depend on class "DefaultChannelScheduler".
