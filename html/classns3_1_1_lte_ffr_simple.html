Detailed Description

Simple Frequency Reuse algorithm implementation which uses only 1 sub-band. Used to test Downlink Power Allocation. When Simple FR receives UE measurements it immediately call functions to change PdschConfigDedicated (i.e. P_A) value for this UE.
