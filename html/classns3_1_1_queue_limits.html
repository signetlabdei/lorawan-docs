Detailed Description

Abstract base class for NetDevice queue length controller.

QueueLimits is an abstract base class providing the interface to the NetDevice queue length controller.

Child classes need to implement the methods used for a byte-based measure of the queue length.

The design and implementation of this class is inspired by Linux. For more details, see the queue limits Sphinx documentation.

◆ Available()

virtual int32_t ns3::QueueLimits::Available ( ) const
pure virtual

Available is called from NotifyTransmittedBytes to calculate the number of bytes that can be passed again to the NetDevice.

A negative value means that no packets can be passed to the NetDevice. In this case, NotifyTransmittedBytes stops the transmission queue.
