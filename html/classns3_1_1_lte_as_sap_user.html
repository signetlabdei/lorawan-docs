Detailed Description

This class implements the Access Stratum (AS) Service Access Point (SAP), i.e., the interface between the EpcUeNas and the LteUeRrc In particular, this class implements the User part of the SAP, i.e., the methods exported by the EpcUeNas and called by the LteUeRrc.
