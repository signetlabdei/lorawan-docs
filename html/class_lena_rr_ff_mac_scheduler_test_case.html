Detailed Description

This system test program creates different test cases with a single eNB and This system test program creates different test cases with a single eNB and several UEs, all having the same Radio Bearer specification. In each test case, the UEs see the same SINR from the eNB; different test cases are implemented obtained by using different SINR values and different numbers of UEs. The test consists on checking that the obtained throughput performance is equal among users and matches a reference throughput value within a given tolerance.
