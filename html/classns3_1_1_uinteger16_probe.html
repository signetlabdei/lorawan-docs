Detailed Description

This class is designed to probe an underlying ns3 TraceSource exporting an uint16_t. This probe exports a trace source "Output" of type uint16_t. The Output trace source emits a value when either the trace source emits a new value, or when SetValue () is called.

The current value of the probe can be polled with the GetValue () method.
