Detailed Description

Make sure that when changing the fragmentation threshold during the simulation, the TCP transmission does not unexpectedly stop.

The scenario considers a TCP transmission between a 802.11b station and a 802.11b access point. After the simulation has begun, the fragmentation threshold is set at a value lower than the packet size. It then checks whether the TCP transmission continues after the fragmentation threshold modification.
