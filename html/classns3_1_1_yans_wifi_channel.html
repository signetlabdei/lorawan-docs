Detailed Description

a channel to interconnect ns3::YansWifiPhy objects.

This class is expected to be used in tandem with the ns3::YansWifiPhy class and supports an ns3::PropagationLossModel and an ns3::PropagationDelayModel. By default, no propagation models are set; it is the caller's responsibility to set them before using the channel.
